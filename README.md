# Ai-health-care
# 🧠 AI Symptom Checker

An AI-powered healthcare assistant that analyzes user symptoms and provides:

## 🚀 Features
- Chatbot-based symptom input
- AI-generated possible conditions
- Urgency detection (Low / Medium / High)
- Medicine suggestions with timing
- Diet recommendations (Eat / Avoid)

## 🛠 Tech Stack
- Frontend: HTML / React
- Backend: Flask (Python)
- AI: OpenAI API

## ⚠️ Disclaimer
This is not a medical diagnosis. Consult a doctor for serious conditions.

## ▶️ How to Run

### Backend
pip install -r requirements.txt  
python app.py  

### Frontend
Open index.html OR run React app
